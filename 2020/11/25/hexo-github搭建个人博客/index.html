hexo+github搭建个人博客

Created2020-11-25|Updated2020-11-26|其他
|Post View:

写在前面

需求软件

nodejs:cmd输入node -v查看安装是否成功
git
可选项-修改npm源:cmd输入npm install -g cnpm –registry=https://registry.taobao.org

正文

一、本地

  1. 安装hexo：cnpm install -g hexo-cli
  2. 新建一个文件夹作为本地文件保存地方（如e:\blog）
  3. 初始化博客：hexo init e:\blog
  4. 浏览器输入 localhost:4000可以查看生成的本地博客
    error:如果打不开链接，可能是端口4000被占用，更改端口地址：在e:\blog文件夹下打开git bash,输入npm install hexo-server安装服务端,再输入hexo server –p 5000将端口地址改为5000，安装服务端后可以在git bash通过hexo -s查看hexo的端口，或者直接关掉其他软件就是了

二、部署到远端

  1. 安装git插件：cnpm install hexo-deployer-git –save
  2. github新建一个和你名字一样的仓库
  3. 记事本打开_config.yml文件
    找到#deployment(注意冒号后面有空格，不能缺省)
    添加：
    type: git
    repo: (你的github地址)
    branch: master
  4. 输入git config –global user.email “你的git账号邮箱”和git config –global user.name “你的git账号名字”
  5. 输入hexo d
